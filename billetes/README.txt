Imágenes de billetes colombianos
================================

Coloque aquí una imagen por cada denominación para que cada casilla muestre el billete correspondiente.

Nombres de archivo (use .png o .jpg):
- 5000.png  (o 5000.jpg)  → billete de $5.000
- 10000.png (o 10000.jpg) → billete de $10.000
- 20000.png (o 20000.jpg) → billete de $20.000
- 50000.png (o 50000.jpg) → billete de $50.000

Si no hay imagen, se mostrará el color de respaldo según la denominación.

Puede usar fotos o ilustraciones de los billetes (respetando derechos de autor). Las imágenes se ajustan automáticamente al tamaño de cada casilla.
